# Blog Platform with Comments

## Overview

Blog Platform with Comments is a responsive web application developed using HTML, CSS, and JavaScript. It allows users to create, edit, delete, and search blog posts while also supporting likes and comments. All data is stored using browser local storage for a seamless user experience.

## Features

- Create Blog Posts
- Edit Existing Posts
- Delete Blog Posts
- Search Posts
- Like Posts
- Add Comments
- Local Storage Integration
- Responsive Design

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Project Structure

blog-platform/

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Open `index.html` in any web browser.
4. Create and manage blog posts.
